Basic

Whatever you search, it has to be expressed so that the result is unique.

You can search for a specific piece of text.

Images can be found using their alt-text.

You can limit the scope by a path which "looks like" a hierarchy to the end user:
foo
sub header
foo
subsub-header
foo

In case all else fails, use data-uitesteri -attribute.
